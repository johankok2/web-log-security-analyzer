## Report Structure

Each generated report includes:

1. **Executive Summary**
   - Total events detected
   - Quick overview of threat landscape

2. **Finding 1: Failed Login Attempts**
   - Detailed IP breakdown
   - Targeted paths and timestamps
   - Severity assessment
   - Actionable recommendations

3. **Finding 2: SQL Injection Attempts**
   - Attack patterns detected
   - Targeted parameters
   - Risk assessment
   - Code remediation guidance

4. **Finding 3: Suspicious User Agents**
   - Tool identification
   - Request analysis
   - Reconnaissance indicators
   - Defensive measures

5. **Access Pattern Analysis**
   - Top 10 most active IPs
   - Traffic volume analysis
   - Anomaly detection

6. **Conclusion**
   - Priority actions
   - Risk summary
   - Next steps
